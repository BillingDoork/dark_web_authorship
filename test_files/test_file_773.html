USPS tracking is not updated in real time. If you recieve a Federal Express or DHL package and track it a few minutes later it will show as delivered.

USPS tracking seems to be updated overnights. So if you view the tracking today you will see it was transited through a facility yesterday, check it tomorrow and you will see it is delievered today.
