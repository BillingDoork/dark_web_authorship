Title: Re: Shipping larger Quantities
Post by: Yousaf on July 13, 2013, 10:22 pm
There is a 13 oz rule with USPS, if your package is over 13 oz in weight you have to take it to the counter and ship it with cash, obviously park far as fuck away in a residential street.  If it is under 13 oz in weight and you have to use the box then you can drop it in a blue USPS box.
