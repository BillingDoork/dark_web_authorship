Title: Dropping more then one package in blue mail box at once?
Post by: antigrid on July 22, 2012, 09:00 am
For the vendors out there, what are your thoughts on mailing multiple orders out at the same time from the same blue mail box drop location? For instance, let's say I am mailing two orders at once, obviously outer packaging is the same, same label format, and both would have correct amount of postage stamps for the flat rate box/envelope. Would this look suspicious? Should they both be from the same return address since it will be obvious they are being sent from the same person being that they have identical labels, packaging, and the use of actual postage stamps?

Seems like it makes sense, but then again using the same return address links them. Is the only option to drop them off at separate locations? Are there certain drop locations where this would be less obvious when sorted (either due to volume or some other factor)?

Title: Re: Dropping more then one package in blue mail box at once?
Post by: awakened350 on July 26, 2012, 12:09 am
There really seems to be a lot of paranoia over packages looking off.  I think for the most part people are just hyper aware of anything that could be a flag when in reality legitimate packages may often have the same flaw. It definitely pays to be careful but I think if we start trying to avoid every possible flag our packages are likely to start to stand out.

How many businesses use stamps but package their items 100% perfectly every single time? The businesses that have packaging perfected are not likely to be the ones using stamps. I'd imagine a package that avoids every single flag could be a flag in itself ;)

Just think about how a small one man legitimate business(the kind of business most likelt to be sending stamped packages) would ship their packages and do it the same way and you'll blend in. I'd certainly avoid using the same blue box over and over because it would be easy to do a stake out that particular mailbox. But running around town to many diff boxes is likely to look much more suspicious than someone dumping a bunch in one.
