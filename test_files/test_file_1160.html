Unfortunately, the post office not scanning tracking labels is a very common problem. There's an entire thread devoted to post office ineptitude in scanning tracking labels on the ebay forums. It's entirely possible that your tracking label won't be scanned until the package arrives at the destination post office. So the only tracking you'll see is for actual delivery.

13 Ounce Rule applies only to stamped(as in postage stamps)mail that is dropped in a USPS blue box.  If you're not dropping your packages there would be no reason to use your own Label 400.  If you do drop you can definitely use labels you procured on your own.

"Delivery status information is not avail. for your item via this web site" <---- If you are getting this massage when trying to track a domestic package you are most likely entering an incorrect tracking number.  The error message pertains to pieces of international mail that can only be tracked via EMS tracking.  It would be normal if you tried to track international mail that didn't support tracking.
