Unlike some countries such as the U.S, Australia (Australia Post) has no knowledge of who lives at any particular address at any given time, nor do they need to know. I'm not absolutely sure, however, the U.S.P.S knows who is living at any address and could cross check information if requested. Why they need to know I'm not sure but it has been mentioned here numerous times previously. I use a few names at the one address and have successfully made over 50 international purchases without any problems (touch wood!). Just make sure the name you use (if that person had  previously lived at your current address) hasn't re-directed the mail to a different address. You can use an altered version of your real name too if you are worried, however all in all, using a fake name is not a problem at all in Oz. :) :)
