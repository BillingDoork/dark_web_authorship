You have to show official ID to get either kind of mail box--the ones at the post office as well as the ones at Mail Box USA type-places. Before 9/11, it was way easier to get those without state-issued ID, but now it's the same requirements. So if that's the reason you want to go non-USPS, it doesn't matter. You'll still have to use your real name.
