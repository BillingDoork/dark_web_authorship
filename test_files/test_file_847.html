Post by: oldtoby on March 24, 2013, 08:17 am
Well look, if you normally get legit packages, a number of illicit ones shouldn't make a difference, if they're stealthy.

Here's the problem - if you're faced with a screw up (nonexistent stealth, torn package, address and/or name errors) and it gets one of your special packages opened, you'd better hope it's the last one, because the p.o. may set aside others inbound to you for inspection. And sure, while you have the same ability to deny one package as a half dozen, the "Gosh, I don't know who's sending me drugs" excuse sounds less and less believable (from a starting position of only maybe just) when the value of the incoming drugs goes up and up and up. Yeah, maybe some old buddy decided to send you an 1/8 in the mail, sure. But *also* some pills, and some shrooms, and some crystal? Unasked for, unordered? Uh-huh.

It's your stress level. But this seems to me an avoidable risk. What's the rush?
