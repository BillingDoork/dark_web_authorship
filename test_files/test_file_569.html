Title: Re: Need help with PGP and address
Post by: acider on January 09, 2013, 01:31 am
Quote from: worldpeace on January 09, 2013, 01:19 am
Hey men need help. how can I add the seller key into gpg4usb?

As the guide I posted earlier describes, you go to the vendor's main page (eg http://silkroadvb5piz3r.onion/silkroad/user/da1293a8c7) and copy the public block.

"Highlight a public key in an email or forum post, right click and select "copy" to put it in your clipboard.
Important: make sure you highlight the entire beginning and ending lines, with all five dashes on either side of the text:

-----BEGIN PGP PUBLIC KEY BLOCK-----

-----END PGP PUBLIC KEY BLOCK-----

A common mistake is to miss the first or last dash when highlighting. PGP programs won't recognize a public key block without all the dashes.
After copying a public key to the clipboard and selecting Import Key -> From Clipboard, you'll see some details about the key. Click OK to finish importing."

If you can't find that block then he doesn't use GPG and you have to send your address not encrypted, or not do business with him at all.

Hey I just downloaded gpg4usb how to use it? I created a new file, entered my adress, selected the default key on the right that cames with the program and then I pressed encrypt button. that's it? I copy that code and then the seller will see my adress? let me know please.

No. The default key that comes with the program belongs to one of the program developers. You have to copy and paste the vendor's PGP key from their vendor page to the left-hand (editor) pane in GPG4USB, and then click on Import key, and choose Editor.

The vendor's key will then appear in the right-hand pane.  Write your address in the left hand pane, put a check in the box beside the vendor's key, and click the Encrypt button. Your encrypted address will now appear in the editor pane.
