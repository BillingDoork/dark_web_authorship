Title: Re: concerned with USPS new changes
Post by: hyperocity on January 29, 2013, 02:39 pm
Well, remember that USPS clerks have been just as in the dark as we've been about the new changes. They're just grunt workers and most of them probably could care less about the fact that illicit substances are mailed daily. They're just paid a small wage to handle the packages, and as long as they're not harmed in the process, they'll keep on doing things the way they're used to until they get yelled at for "doing it wrong" so as I had speculated, it's likely that they may still scan the lime green DCN labels for a while yet.

Just realize that not all packages may get scanned, or that one or all might potentially be returned to sender at any moment. So there's still going to be a level of risk involved if you opt to continue using the green labels. If you choose to do so, I'd recommend using a different return address on each package you send, so they don't get the idea to take a whole bunch of packages with the same return address and personally return them to the house to explain the new policy to some unsuspecting owner. More variety of return addresses == bigger PITA to return them all... Just a thought!
