Probably won't be a popular opinion but you shouldn't be ordering large amounts of weed in ways that will require it to be potentially x-rayed. Weed is a pain in the ass for transporting large amounts in nearly every way, its bulky, it smells (easy to fix but the others aren't) and customers don't like it being compressed. It's also hard to disguise & easy to visually identify.

The best advice with weed is if you need large amounts, grow it yourself or buy it domestically so it never gets near an x-ray. That said there probably are a few ways you could send large amounts on a regular basis but they would all require a lot of setting up which takes capital and really if you've got money to spare doing that you might as well just set up a commercial grow op instead or just getting local contacts.
