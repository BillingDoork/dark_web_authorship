Post by: th3creeper on August 24, 2012, 07:06 pm
Find out from the shipper if it was sent with a signature required for delivery...hopefully not.  This way you simply refuse to sign for anything.  Be aware LE won't necessarily need you to sign for a package to charge you with receiving contraband via the mail.  Also, they are now using tracking bugs slipped inside the pkg so even if you open it 1 week later..it won't really matter.
