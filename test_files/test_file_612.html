I would personally use my real name if i was having something posted to my address it looks a lot less suspect if your name is on the package rather than some completely made up alias, you can always deny you ever ordered the package and it is usually a good idea to write 'return to sender' on it and leave it a couple of days before opening. Having a fake name on your package just leads to more problems and questions from LE.
