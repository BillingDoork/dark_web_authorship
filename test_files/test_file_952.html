Title: Re: Shipping Letter Mail..Advice needed
Post by: Green Camel on April 10, 2013, 01:08 am
The pills will get crushed in an envelope. I never shipped them, so can't help you with it. For powders, see below.

Cut off the ziplock, heat seal the bag, spread the powder across the baggy as evenly as possibly, make a pinhole, put in a vacuum sealer baggy, vacuum seal it, tape it between two sheets of triple-folded paper and put in the centre of a DL envelope.

If you're putting it in a MBB instead of vacuum sealing, don't bother with the pinhole. Also, if you're using a cheap 70gsm envelope, use printed paper rather than plain.
