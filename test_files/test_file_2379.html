Title: Re: Does anyone know how I can attach a picture to a message/send it over SR?
Post by: gromble on October 03, 2012, 09:19 pm
Encrypt the photo file (you _are_ encrypting on this site, right?) and send it just as you would any other message.

Using gpg on the command line that would be:

gpg --armor --recipient RecipientsName --output encryptedJpg.gpg --encrypt photo.jpg

Then copy paste what's in encryptedJpg.gpg into the box and hit send.

The person on the other end decrypts it to a .jpg file and opens it.

You'll want to get rid of the photo's EXIF data, too, just in case. Google it, there's services online that'll do it for you and you might already have software that'll do it.
