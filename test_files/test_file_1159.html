Title: Re: Priority shipping question
Post by: Shroomeister on July 09, 2012, 03:50 pm
USPS Priority does NOT have to be signed for.

Delivery Confirmation (which is an add on service to Priority) does NOT have to be signed for either.

Only time I THINK you have to sign is if the mail is "certified".... there may be other conditions, but nothing regarding priority.

Priority is just faster. AVG 3 days coast to coast.
