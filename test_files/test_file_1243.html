Post by: BlackIris on June 09, 2013, 12:02 pm
If you want to ship it I think the best way is to use a box of some product/object that has about the same size and weight of what you want to ship and packing it as a transaction in ebay or similar. Try to hide as much as possible the fact that it is not the object that should be inside the box and treat it as you are selling the same as used (so, in this case I always took care that the box was not seemingly too new but as an used box would have been; if instead you want to let pass the object as new the best approach is to use a name of a company that usually sells those products to ship it - this can even be better, assuming you can do it).

I've sent myself many times things of high weight that were not exactly what they looked outside and all did go well. True, I am not in the US so I don't know exactly how things works there but nobody here ever checked inside to see if inside there was exactly what advertised in the box if the package seemed legit (i.e. about same weight and size the thing advertised in the box would have). I always packed the thing in a way that it didn't move minimally plus covered the box in wrapping paper all around and all did go good. For how to send it use the usual security parameters for letters (as a legit return address not tied to you, etc.)

But if you want to be extremely sure that everything will go correctly the only way is surely to drive it there. Pay someone to do that for you in case.
