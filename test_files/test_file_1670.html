Title: Re: "Processed Through Sorting Facility"?
Post by: gracevan3569 on July 16, 2012, 04:39 pm
It goes through a sort facility on the sender's end and then through a sort facility on your end (where it will be sent to your route carrier). The location of the sort facility should have been listed (and redacted by you). Was it the initial or end sort facility? If it was the last stage, I would assume you should be receiving soon. If this was international, I'm not familiar with the particulars of that, but assume it can take some time.

My experience domestically had been with Priority mail with a DCN.  I was able to see an update a day. There was an issue with my package so I got to see a lot of days of updates before eventually receiving but still. Even if there was an issue with the address, I could see that through the update.
