Title: Re: Shipping to college dorms
Post by: TNS on August 14, 2013, 05:09 am
There's usually nothing wrong with college post units.

The big deal is the address...

College post offices are typically Contract Postal Units. They are typically given their own zip code and when addressing packages it is very important to follow the address format the college specifies.

Typically it goes sort of like:

Attn: Addressee
College Name
Box Number
1 Example Street
Example, EX 00000 + (box#)

Each college is different and make sure you send some legit packages through the mail to make sure there aren't any troubles with things coming up missing for whatever reason. Always check your box and pick up your packages as soon as possible in the event of a conspicuous package. Always make sure the Addressee is the box holder. If the name does not match many automatically return the package.

I've shipped to them and never had a problem.
