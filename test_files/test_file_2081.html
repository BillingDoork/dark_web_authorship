Title: Re: International shipping from US
Post by: th3creeper on August 22, 2012, 08:29 pm
Yes, you can quite easily providing you pack properly.  Parcels containing "documents of no value" require no customs declaration.  Pack your goods flat so the pkg appears to be nothing more than documents.  Send First Class ....use stamps for postage and drop as you would any domestic letter.  Use the PO's APC to calculate correct postage and don't over stamp.  Print out the return and shipping address if at all possible

PS..No charge my friend.
