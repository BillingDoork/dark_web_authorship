Title: stale ship dates (USPS Click N Ship)
Post by: jakers on December 21, 2012, 07:32 pm
Hey guys, how strict is USPS if you use a label that is more than a day old?  It says you can print the label up until midnight, but that means it won't be collected until the next collection day anyway, if you do wait that long.  However at USPS.com under Postage Meters, it says a couple of things:

1) mail will be returned if it is stale
2) if stale, it will only be returned if a human/carrier notices it (the machines don't notice)

However that's postage meters, not clicknship.  For ClickN Ship under refunds, it says that stale labels "should not be used" -- but doesn't say what happens if you do use them..   I got a doofus who used his credit card to buy a clickn ship label, then used a fake return address (a post office), and then waited 5 days to ship -- supposedly he is going to ship today, but didn't read my urgent messages about printing out a new label.  A quick google ship does say that in some areas, USPS is cracking down on stale labels.
