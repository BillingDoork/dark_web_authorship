To be perfectly honest I don't think it makes all that much difference what name you put on the package for the following reasons:
a) If it gets intercepted, and they do a controlled deliv - it's still you who's holding the package
b) If you're careful about how you buy your coins, and how you access the tor network (i.e. using the latest updates etc), it's really plausibly deniable that it was you who ordered the package. As long as you don't crack under pressure, how would they possibly prove it was you who ordered it? I would open mail addressed to me, and if I felt suspicious about it, i would chuck it in the rubbish/even report that someone had been sending drugs to my address.

Don't stress too much. I'm sure plenty of people use other people's addresses as 'drop points' - and I'm sure the people who live there are afforded some kind of legal protection.

Good luck and happy buying!
