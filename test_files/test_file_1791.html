Title: Re: Any clear answer on this tracking error?
Post by: th3creeper on September 27, 2013, 05:02 am
Quote from: LordofSpeed on September 27, 2013, 03:43 am
Delivery status information is not available for your item via this web site.
A return receipt after mailing may be available through your local Post Office."


This error message ordinarily means one of two things.  Either the tracking number is for an incoming International order that has yet to reach US Customs....or you are punching in a tracking number that does not exist in USPS data base(ie: an invalid set of digits.)  Assuming the number is valid and it is an Int'l order...you can check the tracking status via EMS  (track-trace.com)  Once it arrives on US soil and is processed by Customs...the number will show in USPS system.
