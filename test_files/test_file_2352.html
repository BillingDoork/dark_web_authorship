Title: Re: UK shipping abroad
Post by: PlutoPete on September 30, 2012, 09:14 pm
Get the leaflet RMOP1 from the post office, it contains all the prices and weights and international zones.
Buy your stamps in bulk in varying denominations so you can always apply the correct postage, it's not sus to use multiple stamps but it is sus to overpay.
Make sure you affix a customs sticker to countries that require it, along with a blue airmail sticker and a return address.
