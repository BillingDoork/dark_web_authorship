PO boxes require you give the post office two forms of ID to open one up. They will take down your address in the process. So if a package were to get intercepted. You would be fucked regardless....

IMO PO boxes are a waste unless you have special circumstance's such as....you live with your parents or untrustworthy roommates that could possibly open up your package.

Using fake ID's wouldn't help also because say you go in to pick up a package and you're about to be set up for a CD (Yes you can have a CD in a PO Box - Post office will send you a note)....You'll not only get charged for the dope. You'll get charged for fraud for using the fakes.
