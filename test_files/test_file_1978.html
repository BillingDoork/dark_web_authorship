Yeah, I have to agree at least partly with kmfkewm that this "failed CD attempt" story sounds a bit fishy in a couple respects.

The way any piece of mail gets recorded as "delivered" into their database is by the carrier scanning the bar code on the package with a portable scanner which all carriers carry.  If the carrier scans it as "delivered" he obviously thinks he is delivering to the correct address. If a delivery is made incorrectly there is no way for the system to know what that incorrect address was.  The incorrect receiver of that mail would have to return that package to the PO or back to the carrier before it could be put back into the postal delivery system.  Mis-delivered mail that is returned to the PO is put back into the system for redelivery.  There is no accounting for the address the piece was mistakenly delivered to even if it were Express mail which receives the highest handling priority.  It just doesn't make sense that the PO gave you  the address to which it was wrongly sent.  So the obvious question is how were you able to look up the address the pkg was first (allegedly) delivered to?
