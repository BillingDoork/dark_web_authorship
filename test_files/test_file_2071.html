Was it sent Priority with DCN?  If so, the seller by now is obligated to provide you with that number so you can check to see if there is any scan activity.   Quite honestly, if that seller shipped that order there is no way for him/her to know any more about that pkg than you do(which is nothing)  Your suspicions are not unfounded since it should have arrived by now no matter how it was sent.   There are several factors that might explain why it hasn't arrived but for now all you can do is eliminate some of them and narrow it down.  I don't think a legitimate seller would be so cavalier and quick to dismiss your concerns.  Find out if there is a Delivery Confirmation number.  If yes..check it.  If not...ask the person how they know it is "on the way".

some possible reasons your mail hasn't arrived yet..
missent
misdelivered
seized
incorrectly addressed
returned to sender
stolen
just plain lost somewhere in the system
or...
maybe it was never even sent
