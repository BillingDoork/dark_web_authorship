Title: Shipping the biggest security concern for buyers
Post by: v3n on September 03, 2012, 05:13 am
Receiving the package in the mail is, in my opinion, the biggest security risk with this whole process.

Turning cash into BTC anonymously - easy. very little risk
Logging into the marketplace communicating with encrypted messaging, self destructing encrypted messages if necessary and purchasing goods with untraceable currency - easy (thanks to all who make this possible)

Giving someone an address, even if it is a friends still could easily get you caught up. You're putting a lot of trust in a vendor who you do not know or CAN not trust. If you are buying and caught up this will be how they get you. Everything else in place is rock solid.

All I can think of is a fake ID to get the PO Box then a disguise when you go get the package and only get a few shipments before new fake ID and PO Box in a new location.
Even that I am sure they can still get you on one of those few deliveries.


Thoughts? Comments?
