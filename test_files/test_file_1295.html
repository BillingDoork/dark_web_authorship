Post by: YOABC123 on June 22, 2013, 03:19 am
If it's on time and there is no delays in time-frame from the sending of the mail to the time it was delivered, then just go in and pick it up.

2 days is normal from express from Canada to US, yes? It's when you all of a sudden get an express delivery on day 15 and a note left to go sign for it that I would be worried about any type of controlled delivery attempt.

I say this because if there is no noticeable delays then you have to think to yourself that if the package was intercepted, an investigation would have to be started by the authorities and there would be a lot of red tape which would take (days) to organize and setup.

You will be fine, go for it!
