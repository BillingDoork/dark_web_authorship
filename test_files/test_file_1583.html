Title: Re: Best language to ship ?
Post by: waiguomei on August 12, 2013, 07:16 pm
English and French are the international postal languages.  All post offices everywhere in the world are supposed to be able to cope with a package addressed in either of these two languages.

If you are shipping from a country which is neither English speaking nor French speaking to a country that speaks a language different from the language spoken in your country then it is best to use English or French.

If you are shipping from a country which is neither English speaking nor French speaking to a country that speaks a language the same as the language spoken in your country then it is best to use the language which is common between the two countries.

If the buyer is from a country which is neither English speaking nor French speaking and the buyer has specifically given you their PGP encrypted address in bilingual format (English or French + native language) then you should address it bilingually.
